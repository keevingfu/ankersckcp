
### 2. 生成组件代码 (2 分钟)
```bash
# 示例: 从 Figma Button 生成 React 代码
claude --mcp figma-desktop get-component "Button" | \
claude --mcp magic-ui generate-component \
    --framework "react" \
    --output "Button.tsx"
```

### 3. 导出设计资源 (1 分钟)
